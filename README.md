# ED_Lista01.06
## Soma Sem Operador de Soma

Este desafio consiste em criar uma aplicação, em Java, que some dois números naturais sem utilizar o operador de adição.

```
Requisitos do projeto:

1. Arquitetura MVC;
2. A soma deve ser realizada por uma função recursiva;
3. Não é permetido usar métodos de bibliotecas que sejam usados para soma(Math...., i++,...);
4. Declarar, via comentário, a condição de parada da função.
```
